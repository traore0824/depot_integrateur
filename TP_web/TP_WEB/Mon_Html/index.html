Services de qualité

la qualité de service évoque un engagement sur un niveau de qualité donné pour une prestation. L'objectif est de valoriser une vente par l'offre de prestations attendues pour chaque client : accueil, conseil, SAV. Cette 2e dimension est fréquemment visible dans les offres liées aux infrastructures informatiques. On retrouve d'ailleurs l'acronyme QoS pour qualifier ce type d'engagement.
